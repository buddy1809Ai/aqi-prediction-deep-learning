# Data

## CPCB Multi-City Air Quality Dataset

Full dataset: ~18.7 million 15-minute records across 19 Indian cities (2017–2023).

**Full dataset is NOT included in this repository** due to size (~700 MB cleaned).

### Download Full Dataset
- Contact CPCB: https://cpcb.nic.in/
- Or request from the study authors via the contact in README.md

### Samples
The `samples/` folder contains the first 500 rows of each city's cleaned dataset
for demonstration and testing purposes.

### Cities Covered
Ahmedabad, Bhopal, Chennai, Delhi NCR, GandhiNagar, Hyderabad, Indore,
Jaipur, Jodhpur, Mumbai, Mumbai Suburbs, Nagpur, Navi Mumbai, Pune,
Singrauli, Surat, Thane, Vapi, Vishakhapattanam
